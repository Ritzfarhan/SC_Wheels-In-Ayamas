	            
		===> Installation Manual <===

          1. Using Live Server in Visual Studio Code

	  i) Open Visual Studio Code.
	     - Open your VSCode editor.

	 ii) Create basic code file in the editor.
	     - Create index.html and use emit abbreviation to generate HTML5  basic template in visual studio code.

	iii) Go to Extension in VSCode and search live server by Ritwick Dey.
	     - Click on install to add the extension to your editor.

	 iv) Right click on index.html and select open with Live server.
	     - The web page will now open on your PC’s default web browser.
	     - Live server opens in port number 5500.

	  v) You can also use the visual studio code Live Server shortcut option.
	     - There is a shortcut option at the end of the editor screen called “Go Live” 

           2. Node.js

	   i) Download Node.js Installer
	  ii) Once the installer finishes downloading, launch it. Open the downloads link in your browser and click the file. Or, browse to the location where you have saved the file and 		      double-click it to launch.
	 iii) The system will ask if you want to run the software – click Run.
	  iv) You will be welcomed to the Node.js Setup Wizard – click Next.
	   v) On the next screen, review the license agreement. Click Next if you agree to the terms and install the software.
	  vi) The installer will prompt you for the installation location. Leave the default location, unless you have a specific need to install it somewhere else – then click Next.
	 vii) The wizard will let you select components to include or remove from the installation. Again, unless you have a specific need, accept the defaults by clicking Next.
	viii) Finally, click the Install button to run the installer. When it finishes, click Finish
